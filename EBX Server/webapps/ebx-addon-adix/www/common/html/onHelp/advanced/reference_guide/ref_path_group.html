Path domain

Function

The path domain contains the declaration of all paths used in the data mapping configurations. The Path table collects all the path declarations used in the data mapping configuration.

Property

Definition

Application type

A reference to the application type (EBX, Default SQL, XML and XML).

Path type

Either Table, Field or Application.

Path

The path value.

SQL data source

Records of SQL data source table which define the data model and the corresponding external database information.

Comment

Description of the path.
