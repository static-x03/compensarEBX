Introduction 

TIBCO EBX is an all-in-one, model-driven, agile and
multi-domain solution to manage, govern, and consume shared
data assets.

Delivery of TIBCO EBX includes a component called TIBCO EBX
Add-ons.  Please see the TIBCO EBX Add-ons readme for important
licensing information.

If you are licensed for TIBCO EBX(R) GO, you are authorized to
use TIBCO EBX(R) GO for Android and TIBCO EBX(R) GO for iOS
mobile applications.  TIBCO EBX GO for Android is available
through Google Play and is subject to its own license terms. 
TIBCO EBX GO for iOS is available through Apple App Store and
is subject to its own license terms.  Usage of these two mobile
applications requires the TIBCO EBX(R) GO Add-on which is
licensed as part of TIBCO EBX(R) Add-ons.  Please see the TIBCO
EBX Add-ons readme for additional information.

If you are licensed for TIBCO EBX(R) D3, you are authorized to
use an EBX instance with its usage restricted as a replica. 

If you are licensed for the TIBCO EBX(R) Enterprise Bundle, you
are authorized to use TIBCO EBX with the following Premium
Add-Ons:
-- TIBCO EBX(R) Match and Cleanse Add-on
-- TIBCO EBX(R) Data Exchange Add-on
-- TIBCO EBX(R) Insight Add-on
