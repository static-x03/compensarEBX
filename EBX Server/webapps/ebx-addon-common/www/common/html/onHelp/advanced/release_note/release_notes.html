Version 3.0.0

Released: March 2021

Featured update

The add-on has been updated to ensure compatibility with the TIBCO EBX® 6.0.0 release.

Release Note 2.1.12

Release Date: March 2021

Updates

An update was applied to correct an issue with the Apache Standard Taglibs library.

Release Note 2.1.11

Release Date: February 2021

Bug fixes

This release includes support for bug fixes in the TIBCO EBX® (version 5.9.13).

Release Note 2.1.10

Release Date: January 2021

Featured updates

This release contains the following library updates:

  • Apache Standard Taglibs library to version 1.2.3.

  • Spring framework library to version 5.2.9.

  • Jackson Databind library to version 2.11.2.

Release Note 2.1.9

Release Date: September 18, 2020

Featured update

The add-on has been updated to support the OpenJDK8 and OpenJDK11 libraries.

Bug fixes

[COMM-100] Libraries were updated to fix some potential issues.

Release Note 2.1.8

Release Date: June 23, 2020

Featured update

Support for JDK8 and JDK11 has been updated. Additionally, the jackson, slf4j, and jQuery libraries have been updated.

Bug fixes

[COMM-94] An add-on description in French is not translated.
