Custom data model overview

With the TIBCO EBX® Data Model and Data Visualization Add-on you can display data models not included in your EBX® repository. The API contains the classes and methods for all components required to generate a data model graph. So, you can write a little Java code, make some minor configuration adjustments, and use the add-on to render a graph.
